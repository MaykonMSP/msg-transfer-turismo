# MSG Transfer Turismo - Landing Page

Landing page profissional desenvolvida para a **MSG Transfer Turismo**, empresa de transfer, turismo, city tour e passeios na Serra Gaúcha.

O projeto foi criado com foco em apresentação comercial, captação de contatos via WhatsApp e uso em campanhas de divulgação, incluindo tráfego pago.

## Objetivo Comercial

O objetivo da página é apresentar os serviços da MSG Transfer Turismo de forma elegante e confiável, destacando roteiros, diferenciais, imagens reais da região e chamadas para contato direto pelo WhatsApp.

A landing page foi pensada para:

- Gerar solicitações de orçamento
- Facilitar contato via WhatsApp
- Apresentar serviços de transfer e city tour
- Valorizar destinos de Gramado, Canela e Serra Gaúcha
- Transmitir confiança para potenciais clientes

## Tecnologias Usadas

- React
- Vite
- Tailwind CSS
- Lucide React
- JavaScript
- HTML5
- CSS3

## Funcionalidades

- Hero visual com imagem real da Serra Gaúcha
- Seções de serviços, galeria, roteiros, diferenciais, depoimentos e FAQ
- CTA final com imagem de fundo e chamada para WhatsApp
- Botão flutuante de WhatsApp
- Página interna de Política de Privacidade
- Rodapé com informações básicas de confiança
- Layout responsivo e mobile first
- Animações leves e efeitos de hover
- Estrutura pronta para GitHub e deploy

## Como Rodar Localmente

Clone o repositório e instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Depois, acesse a URL exibida no terminal, normalmente:

```bash
http://localhost:5173/
```

## Build de Produção

Para gerar os arquivos finais de produção:

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta:

```bash
dist/
```

Para visualizar o build localmente:

```bash
npm run preview
```

## Deploy

Este projeto pode ser publicado em plataformas de hospedagem estática, como:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

Configurações comuns:

- Comando de build: `npm run build`
- Pasta de saída: `dist`
